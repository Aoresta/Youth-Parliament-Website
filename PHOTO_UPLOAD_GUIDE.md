# Photo Upload Guide

Upload all member photos inside:

```txt
public/members/
```

After uploading, open `app/members-edit-here.js` and change the member's `image` value.

Example:

```js
{
  name: "Tejas Bansal",
  role: "Finance Minister",
  image: "tejas-bansal.jpg",
  ideology: "Either Go Har(d) or Go Home"
}
```
